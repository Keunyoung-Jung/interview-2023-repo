단어사전 가이드 문서

단어 사전

명사 탭

명사 탭에서는 단어사전에 추가하고 싶은 명사를 입력한다.

영문 또는 한글 단어를 등록할 수 있다.

+추가 버튼을 눌러 단어를 추가할 수 있다.

업데이트가 true 인 경우 사전에 반영된 상태

업데이트가 false인 경우 사전에 미반영 상태

등록하기전, 이미 등록되어 있는지 검색창에 검색하여 확인할 수 있다.

형용사 탭

형용사탭은 형용사로 작성되는 단어를 입력한다.

등록 과정은 명사 탭과 동일하다. 형용사는 ‘다’ 어미를 작성하지 않고, 변형전 상태로 입력한다.

접미사 탭

접미사는 단어의 뒤에 붙어 한 단어가 될 수 있는 단어를 말한다.

예를들어, 불고기소스 단어는 불고기(명사), 소스(명사)로 분리될 수 있다. 하지만 불고기소스는 한국어에서 합성어로써 한 단어로 인식할 수 있어야한다. 그 중 소스는 접미사로 붙어 불고기소스, 미트소스, 머스타드소스 등으로 한번에 처리 하기 위해서 단어 사전에 소스(접미사)로 등록하여 한단어로 인식되게 할 수 있다.

등록과정은 명사 탭과 같다.

맞춤법 수정 탭

맞춤법 수정 탭은, SNS 단어 특성상 오타, 외래어, 줄임말, 한국어 변형 등 원래의 단어로 되돌리기 위해 사용한다.

그러나, 해당 단어를 원래 단어로 변형하여 저장되므로 주의가 필요하다.

예를 들어, 외래어라서 두개의 단어로 분리되는 경우 트레져 = 트레저 로 변형하도록 입력할 수 있지만, 야민정음이라 불리는 네넴띤=비빔면 으로 등록하는 경우 → 네넴띤 이라는 단어가 유행임에도 비빔면으로 저장되어 유행 단어를 파악할 수 없게 될 수 있으므로 적절한 단어 등록이 필요하다.

#NIZ 키워드

불용어 탭

NIZ 키워드 불용어 탭은 실시간으로 제거되어야할 단어를 입력할 때 사용된다.

NIZ 서비스에서 보여지는 키워드 중 실시간으로 숨길 수 있는 항목

+추가 버튼을 눌러서 단어 추가 페이지로 진입

숨길단어를 입력하고 저장 버튼을 누르면, 저장이 완료됨

저장하고 새로운 작업 버튼 누를 시, 저장된 후 새로운 입력창으로 refresh 된다.
